# Base64 工具箱

免费的在线 Base64 编解码工具，支持文本和图片转换。

## 功能特性

- 📝 **文本转换**：支持中文文本的 Base64 编解码
- 🖼️ **图片转换**：支持 JPG、PNG、GIF、WebP 格式图片转 Base64
- ⚡ **快速转换**：纯前端处理，秒级响应
- 🔒 **隐私保护**：所有转换在浏览器本地完成，不上传服务器
- 🆓 **完全免费**：无限制使用，无需注册
- 📱 **响应式设计**：支持手机、平板、电脑多端访问

## 部署方式

本项目支持以下免费托管平台：

### Vercel（推荐）
1. 安装 Vercel CLI：`npm i -g vercel`
2. 运行：`vercel`
3. 按提示完成部署

### Netlify
1. 将代码推送到 GitHub
2. 在 Netlify 中连接仓库
3. 自动部署

### GitHub Pages
1. 创建 GitHub 仓库
2. 上传文件
3. 启用 GitHub Pages

## 本地运行

直接用浏览器打开 `index.html` 即可，无需安装依赖。
